# Industrial_camera_acquisition_software
Industrial_camera_acquisition_software是一款基于 Python 和 PyQt5 开发的高级干涉测量与光束轮廓分析软件。该程序专为光学工程、高功率激光测试以及涡旋光束（如拉盖尔-高斯光束）的特性表征而设计。软件集成了底层相机 SDK 直连、实时 2D 光斑轮廓分析（支持 D4σ、1/e²、刀口法拟合）、$M^2$ 光束质量计算工具，以及一个内置的数学仿真引擎。其核心亮点在于集成了一套实时干涉相位重建模块，旨在通过傅里叶轮廓术 (FTP) 和空域外差解调技术，从离轴干涉图中直接提取光束的波前相位分布。
